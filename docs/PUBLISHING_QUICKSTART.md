# Быстрый старт — Планирование публикаций

## Шаг 1: Подготовка контента

Перед планированием публикаций убедитесь, что у вас есть готовый контент:

1. Импортируйте товары (кнопка "Импортировать товары")
2. Для каждого товара сгенерируйте:
   - Текстовый контент (описания, посты)
   - Изображения (1 изображение)
   - Видео

Только контент со статусом `ready` будет доступен для публикации.

---

## Шаг 2: Подключение социальных сетей

1. Перейдите на страницу "Creators" (`/creators`)
2. Нажмите "Подключить аккаунт"
3. Выберите платформу (YouTube)
4. Пройдите OAuth авторизацию
5. Аккаунт появится в списке подключённых

Можно подключить несколько аккаунтов на одной платформе.

---

## Шаг 3: Планирование публикаций

### Открытие страницы публикаций

Перейдите на страницу "Publishing" (`/publishing`)

### Создание новых публикаций

1. **Нажмите кнопку "+ Запланировать публикации"**

2. **Выберите контент:**
   - Откроется модальное окно со списком готового контента
   - Используйте фильтры:
     - Тип контента: все / видео / изображения
     - Платформа: YouTube
   - Отметьте нужный контент (чекбоксы)
   - Можно использовать "Выбрать все"
   - Нажмите "Продолжить (N)" где N — количество выбранного

3. **Настройте публикации:**
   
   Для каждого выбранного контента:
   
   - **Платформа** (обязательно)
     - Выберите: YouTube
   
   - **Аккаунт** (обязательно)
     - Выберите из подключённых аккаунтов для выбранной платформы
     - Если аккаунтов нет — вернитесь к Шагу 2
   
   - **Дата и время** (обязательно)
     - По умолчанию: +1 час для каждого следующего контента
     - Время должно быть в будущем
     - Формат: дата + время (например, 01.03.2026 15:00)
   
   - **Заголовок** (опционально)
     - Максимум 100 символов
     - Если не указан — будет использовано название товара или автоматически сгенерировано
   
   - **Описание** (опционально)
     - Максимум 5000 символов
     - По умолчанию подставляется текст контента
     - Для длинных видео (≥60 сек) ссылка на товар добавится автоматически

4. **Нажмите "Запланировать"**
   - Система проверит все данные
   - Создаст записи в очереди публикаций
   - Публикации "сейчас" начнут обрабатываться сразу
   - Будущие публикации будут ждать своего времени

## Шаг 4: Управление публикациями

### Просмотр очереди

На странице "Publishing" вы увидите таблицу со всеми публикациями:

- **Content ID** — идентификатор контента
- **Платформа** — YouTube
- **Запланировано** — дата и время публикации
- **Статус** — текущее состояние:
  - 🟡 **Ожидает** — публикация запланирована
  - 🟠 **В процессе** — идёт загрузка на платформу
  - 🟢 **Опубликовано** — успешно загружено
  - 🔴 **Ошибка** — что-то пошло не так
- **Создано** — когда публикация была запланирована
- **Действия** — доступные операции

### Фильтрация

Используйте фильтры для поиска нужных публикаций:

- **Статус:** все / ожидает / в процессе / опубликовано / ошибка
- **Платформа:** YouTube

Нажмите "Сбросить фильтры" для возврата к полному списку.

### Отмена публикации

Для публикаций со статусом "Ожидает":
1. Нажмите кнопку "Отменить" в строке публикации
2. Подтвердите действие
3. Публикация будет удалена из очереди

**Важно:** Нельзя отменить публикации, которые уже в процессе или завершены.

### Просмотр результата

Для опубликованных видео:
1. Нажмите кнопку "Открыть"
2. Откроется видео на YouTube

## Примеры использования

### Пример 1: Публикация одного видео на YouTube

1. Перейдите на `/publishing`
2. Нажмите "+ Запланировать публикации"
3. Выберите одно видео
4. Настройте:
   - Платформа: YouTube
   - Аккаунт: Ваш YouTube канал
   - Дата и время: сегодня, через 2 часа
   - Заголовок: "Обзор товара: [название]"
   - Описание: используйте сгенерированное описание
5. Нажмите "Запланировать"

### Пример 2: Массовая публикация на несколько платформ

1. Перейдите на `/publishing`
2. Нажмите "+ Запланировать публикации"
3. Выберите 5 видео
4. Для первых 2 видео:
   - Платформа: YouTube
   - Время: сегодня 15:00, 16:00
5. Для следующих 3 видео:
   - Платформа: YouTube
   - Время: завтра 10:00, 11:00, 12:00
6. Нажмите "Запланировать"

Система создаст 5 публикаций одним запросом.

### Пример 3: Еженедельный контент-план

Запланируйте публикации на неделю вперёд:

1. Выберите 7 видео
2. Настройте публикации на каждый день:
   - Понедельник 18:00
   - Вторник 18:00
   - Среда 18:00
   - и т.д.
3. Используйте разные платформы для разнообразия
4. Система автоматически опубликует в указанное время

## Troubleshooting

### Проблема: Нет доступного контента

**Решение:**
- Убедитесь, что контент сгенерирован
- Проверьте статус контента (должен быть `ready`)
- Только видео и изображения доступны для публикации

### Проблема: Нет подключённых аккаунтов

**Решение:**
- Перейдите на страницу "Creators"
- Подключите нужную платформу через OAuth
- Вернитесь к планированию публикаций

### Проблема: Публикация со статусом "Ошибка"

**Возможные причины:**
- Токен доступа истёк (обновляется автоматически)
- Файл видео не найден
- Ошибка API платформы
- Нарушение правил платформы

**Решение:**
- Проверьте сообщение об ошибке в таблице
- Попробуйте запланировать публикацию заново
- Убедитесь, что аккаунт всё ещё подключён

### Проблема: Не могу отменить публикацию

**Причина:** Публикацию можно отменить только если она в статусе "Ожидает"

**Решение:**
- Публикации в процессе или завершённые отменить нельзя
- Если нужно удалить опубликованное видео — сделайте это на самой платформе

## Лимиты и ограничения

### Rate Limits
- **Одиночная публикация:** 5 запросов в минуту
- **Массовая публикация:** 3 запроса в минуту

### Ограничения на запрос
- **Максимум публикаций за раз:** 50
- **Длина заголовка:** 100 символов
- **Длина описания:** 5000 символов

### Платформы
- **YouTube:** полная поддержка (загрузка видео)
- **YouTube:** полная поддержка (загрузка видео)

## Советы и best practices

### 1. Планируйте заранее
Создавайте публикации на несколько дней вперёд для стабильного контент-плана.

### 2. Используйте разные платформы
Публикуйте контент на YouTube для максимального охвата.

### 3. Оптимальное время публикации
- **YouTube:** 14:00-16:00, 18:00-20:00
- **YouTube:** 12:00-14:00, 18:00-21:00
- Учитывайте часовой пояс вашей аудитории

### 4. Заполняйте описания
Хорошее описание повышает видимость видео в поиске.

### 5. Проверяйте статус
Периодически заходите на страницу публикаций и проверяйте, что всё идёт по плану.
